# NoGravityZone

## О проекте
Данное мобильное приложение предоставляет необходимый функционал для удобного и эффективного тренировочного процесса по такому виду спорта, как стритлифтинг.

Стритлифтинг — это один из видов силового вида спорта, включающий упражнения: выходы силой, подтягивания, отжимания на брусьях и приседания с дополнительным весом.

## Функциональные возможности
- Регистрация и авторизация пользователя через email и пароль или OAuth (Google);
- Восстановление пароля пользователя;
- Просмотр личного кабинета с персональными данными, целями и истории прогресса в виде статистики;
- Наличие и просмотр необходимого списка оборудования, рекомендаций по питанию, добавок и соревнований по стритлифтингу;
- Создание персональных тренировок c интеграцией с Google Calendar для напоминаний;
- Возможность использования предустановленных программ тренировок. 

## Используемые технологии
### Клиентская часть (Android)
- **Язык**: Kotlin
- **Среда разработки**: Android Studio
- **Архитектура**: MVVM
- **Библиотеки**:
  - Jetpack Compose (UI)
  - Navigation API (навигация)
  - Room (локальная база данных SQLite)
  - Google OAuth API (авторизация)
  - Google Calendar API (синхронизация тренировок)

### Серверная часть (при необходимости)
- **Язык**: Kotlin
- **Фреймворк**: Spring Boot
- **База данных**: PostgreSQL
- **API**: REST для связи с клиентской частью

## Установка и запуск
### 1. Клонирование репозитория
```bash
git clone https://github.com/proxod3-first/nogravityzone.git
cd nogravityzone
```

### 2. Запуск клиентской части
Открыть проект в **Android Studio**, запустить эмулятор или подключить физическое устройство, затем выполнить сборку и запуск.

### 3. Запуск серверной части (при наличии)
```bash
cd server
./gradlew bootRun
```

## Структура проекта
Проект разделен на три слоя:
1. **Presentation** — UI и ViewModel
2. **Domain** — бизнес-логика
3. **Data** — база данных, API, (кэширование)
